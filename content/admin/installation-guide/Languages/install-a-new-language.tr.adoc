== Dil paketi indirme

. https://sourceforge.net/projects/suitecrmtranslations/files/[Stable SuiteCRM Language Packs^] veya https://crowdin.com/project/suitecrmtranslations[Latest Build SuiteCRM Language Packs^] adreslerinden birini seçin.
. Sürümünüzü ve dilinizi seçin
. ZIP dosyasını indirin
