= 12. Logic Hooks

== Giriş

Logic hook'lar SuiteCRM'deki çeşitli olaylara bağlanarak özelleştirilmiş kodun
çalışmasını sağlar. Bu, örneğin harici bir API'ye çağrı yapmak veya belirli bir olay
gerçekleştiğinde yeni bir kayıt oluşturmak gibi işlemleri mümkün kılar.

== Türler

Logic hook'lar üç bağlamda gerçekleşebilir: Application Hooks, Module Hooks ve
User Hooks. Aşağıda detaylandırılmıştır.

ait değildir). Bu hook'lar genellikle `custom/modules/logic_hooks.php` içinde tanımlanır.
