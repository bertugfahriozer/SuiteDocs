== 1. Giriş

Bu rehber, 8.5 öncesi sürümlerde oluşturulmuş ön uç eklentilerini SuiteCRM 8.5+ ile uyumlu hale getirmek için gereken adımları açıklar.

== 2. Güncellenen `@angular-architects/module-federation` kütüphanesi

Angular v16'ya yükseltmeyle birlikte `@angular-architects/module-federation` sürümü de güncellendi. Bu paket, frontend eklentilerinin çalışması için ana kütüphanedir.

Bu yeni sürüm yapılandırma güncellemesi gerektirir; dolayısıyla önceden derlenmiş eklentilerin yeniden derlenmesi gerekir (`yarn run build:defaultExt` gibi).

== 3. defaultExt eklentisini güncelleme

`defaultExt` için yapılandırma yükseltmeye otomatik olarak uyum sağlar. Eksik adımlar:

- `yarn run build:defaultExt` çalıştırın
- `extensions/defaultExt/config/extension.php` içindeki eklentiyi yeniden etkinleştirin

== 4. Özel eklentileri güncelleme (defaultExt dışındakiler)

Özel bir eklentiyi güncellemek için üç dosya güncellenmelidir:

- `extensions/your-extension/config/extension.php`
- `extensions/your-extension/app/tsconfig.app.json`
- `extensions/your-extension/app/webpack.config.js`

{{% notice info %}}
Herhangi bir şüphe durumunda `extensions/defaultExt` içindeki örnek dosyalara bakın.
{{% /notice %}}

=== 4.1 `extensions/your-extension/config/extension.php` güncellemesi

`remoteEntry` içinde `./` yerine `../` kullanın. Örnek:

From `'remoteEntry' => './extensions/your-extension/remoteEntry.js',`

to `'remoteEntry' => '../extensions/your-extension/remoteEntry.js',`
