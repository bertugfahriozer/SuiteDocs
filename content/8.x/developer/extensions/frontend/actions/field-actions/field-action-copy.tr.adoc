== 1. Giriş

Alan eylemleri, SuiteCRM'de alanların yanına özel eylem butonları eklemenizi sağlar.

Copy (kopyala) alan eylemi, alan değerini panoya (clipboard) kopyalamanıza olanak tanır.

Bu rehber, alan kopyalama eyleminin bir alana nasıl eklenebileceğini adım adım anlatır.

image:copy-field-action-example.gif[Alan Kopyalama Eylemi Örneği]

== 2. Mantık (Logic) Metadata tanımı

{{% notice note %}}
Yapacağınız değişiklikleri custom dizininde yapmayı unutmayın; örn: `public/legacy/custom/modules/<module>/...`
{{% /notice %}}

İlk olarak metadata içindeki `fieldActions` girdisini tanımlamalısınız. Alanlar için eylem butonları buradan tanımlanır.

Konfigürasyon yalnızca `detailviewdefs.php` dosyasına eklenebilir.

Aşağıdaki örnekte bunları `Accounts` modülünün `detailviewdefs.php` dosyasına ekleyeceğiz.

=== 2.1 Custom detailviewdefs.php üzerinde eylem ekleme adımları

. Core Accounts `detailviewdefs.php` dosyasını custom klasöre kopyalayın:
.. From: `public/legacy/modules/Accounts/metadata/detailviewdefs.php`
.. To: `public/legacy/custom/modules/Accounts/metadata/detailviewdefs.php`
. Custom `detailviewdefs.php` içinde kendi eylem butonunuz için yeni bir giriş ekleyin (aşağıdaki snippet örneği)
** Gerekirse izinleri yeniden ayarlayın (ortamınıza bağlıdır)
. Admin menüsünden `Repair and Rebuild` çalıştırın

=== 3. Özelliklerin açıklaması

* Key
- `actions` dizisi içindeki `key` hangi eylem tipinin kullanılacağını belirtir. Bu durumda `copy`.

* Modes
- Modes, eylemin hangi görünümlerde etkili olacağını belirtir; örnek: `edit`.

* Icon
