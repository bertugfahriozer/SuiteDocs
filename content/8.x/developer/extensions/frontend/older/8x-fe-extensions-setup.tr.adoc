== Extension framework kurulumu

Bu rehber, bir frontend eklenti modülü kurmak için adım adım açıklamalar sağlar. Bu eklenti yeni frontend özelleştirmeleri eklemenize veya mevcut özellikleri override etmenize olanak verir.

== 1. Eklenti ismi seçimi

Eklenti makine adını seçin; boşluk, `-` veya özel karakter kullanmayın. CamelCase önerilir (örn: `myExt`) ve dosya/klasör adı için snake-case (`my-ext`) önerilir.

*Not:* Bu dosya adlandırma önerisi, case-sensitive sistemlerde sorun yaşamamak içindir.

== 2. Mikro frontend (angular app) oluşturma

. `ng g app <your-extension-name> --projectRoot=extensions/<your-extension-name>/app --routing=false --style=scss` komutunu çalıştırarak oluşturun.
.. Örnek: `ng g app myExt --projectRoot=extensions/my-ext/app --routing=false --style=scss`

== 3. Module Federation başlatma

Extension üzerinde module-federation etkinleştirin.

=== 3.1 Module federation etkinleştirme

. `ng add @angular-architects/module-federation --project=<your-extension-name>` komutunu çalıştırın.
.. Örnek: `ng add @angular-architects/module-federation --project=myExt`
