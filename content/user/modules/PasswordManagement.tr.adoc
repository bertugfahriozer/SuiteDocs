== Parola Yönetimi

Admin panelinden *Password Management* seçeneğini seçerek ayarlar sayfasını açın.

=== Parola Güvenlik Ayarları

Bu seçenekler kullanıcı parolaları için isteğe bağlı güvenlik gereksinimleridir. Bir kez ayarlandığında, kullanıcı parolaları seçili kriterlere uymak zorunda kalır.

image:PasswordSecuritySettings.png[title = "Password security settings"]

Gerekli parola özelliklerini seçmek için kutuları işaretleyin.

Özel karakterler şunlardır: #$%^&*()+=-[]';,./{}|:<>?~

Click btn:[Save] ile ayarları kaydedin.

== İki Faktörlü Doğrulama

İki Faktörlü Doğrulama (Two Factor Authentication) kullanıcı bazında etkinleştirilebilir. Etkinleştirildiğinde, kullanıcı her oturum açtığında e-posta ile gönderilen bir kodu girmek zorunda kalır.

Kullanıcının profilini açın ve *Actions* menüsünden *Edit* seçeneğini seçin.

image:TwoFactorAuth.png[title="Two Factor Authentication - user settings"]

Two Factor Authentication kutusunu işaretleyin.

Click btn:[SAVE] ile ayarları kaydedin.

Kullanıcı oturum açtığında, kullanıcı adı ve parolayı girdikten sonra e-posta ile gönderilmiş olan kodu girmesi istenecektir.

image:TwoFactorAuthCodeSent.png[title="Two Factor Authentication code required"]

Doğru kodu girip btn:[VERIFY] düğmesine tıkladıktan sonra kullanıcı SuiteCRM'e giriş yapmış olacaktır.

*E-posta Şablonu*

İki Faktörlü Doğrulama için gönderilen kod e-postasının şablonu Admin panelindeki *Password Management* bölümünden ayarlanabilir.
